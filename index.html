Headless Browsers

A list of (almost) all headless web browsers in existence

  • PhantomJS - PhantomJS is a headless WebKit scriptable with a JavaScript API. It has fast and native support for various web standards: DOM handling, CSS selector, JSON, Canvas, and SVG.
  • CasperJS - CasperJS is an open source navigation scripting & testing utility written in Javascript for the PhantomJS WebKit headless browser and SlimerJS (Gecko).
  • Zombie.js - Zombie.js is a lightweight framework for testing client-side JavaScript code in a simulated environment. No browser required.
  • Lotte - Lotte is a headless, automated testing framework built on top of PhantomJS and inspired by Ghostbuster.
  • Ghostbuster - Automated browser testing via phantom.js, with all of the pain taken out! That means you get a real browser, with a real DOM, and can do real testing!
  • HeadlessBrowser - A very miniature headless browser, for testing the DOM on Node.js
  • browser.rb - Headless Ruby browser on top of Nokogiri and TheRubyRacer
  • Selenium - Selenium is a suite of tools to automate web browsers across many platforms.
  • SlimerJS - SlimerJS is similar to PhantomJs, except that it runs Gecko, the browser engine of Mozilla Firefox, instead of Webkit (And it is not yet truly headless).
  • trifleJS - A headless Internet Explorer browser using the .NET WebBrowser Class with a Javascript API running on the V8 engine.
  • DalekJS - Automated cross browser testing with JavaScript.
  • HtmlUnit - HtmlUnit is a "GUI-Less browser for Java programs".
  • ghost.py - ghost.py is a webkit web client written in python.
  • twill - twill is a simple language that allows users to browse the Web from a command-line interface.
  • Awesomium - Chromium-based headless browser engine
  • SimpleBrowser - A flexible and intuitive web browser engine designed for automation tasks. Built on the .Net 4 framework.
  • Envjs - Envjs is a simulated browser environment written in javascript.
  • Watir-WebDriver - The most elegant way to use Selenium WebDriver with ruby.
  • Spynner - Programmatic web browsing module with AJAX support for Python
  • WatiN - Web Application Testing In .Net
  • Crowbar - Crowbar is a web scraping environment based on the use of a server-side headless mozilla-based browser.
  • BrowserKit - BrowserKit simulates the behavior of a web browser.
  • SST - SST (selenium-simple-test) is a web test framework that uses Python to generate functional browser-based tests.
  • stanislaw - Python headless browser testing
  • Jabba-Webkit - Jabba's headless webkit browser for scraping AJAX-powered webpages.
  • Jaunt - Java Web Scraping & Automation API
  • Geb - Geb is a browser automation solution.
  • Jasmine-Headless-Webkit - jasmine-headless-webkit uses the QtWebKit widget to run your specs without needing to render a pixel.
  • phantompy - Phantompy is a headless WebKit engine with powerful pythonic api build on top of Qt5 Webkit
  • Python-Webkit - Python-Webkit is a python extension to Webkit to add full, complete access to Webkit's DOM
  • Splinter - Splinter is an open source tool for testing web applications using Python. It lets you automate browser actions, such as visiting URLs and interacting with their items.
  • Browserjet - Headless webkit browser with node.js interface.
  • Headless - Headless browser support for fast web acceptance testing in .Net
  • browser-launcher - detect and launch browser versions, headlessly or otherwise
  • headless_browser - Headless browser based on WebKit written in C++.
  • Guillotine - A .NET headless browser, written in C#.
  • benv - Stub a browser environment in node.js and headlessly test your client-side code.
  • grope - A non-GUI library to represent browser environment using WebKit Framework + RubyCocoa
  • RoboBrowser - A simple, Pythonic library for browsing the web without a standalone web browser.
  • MechanicalSoup - A Python library for automating interaction with websites.
  • mechanize - Stateful programmatic web browsing.
  • jedi-crawler - Lightsabing Node/PhantomJS crawler; scrape dynamic content : without the hassle
  • Splash - Splash is a javascript rendering service with an HTTP API. It's a lightweight browser with an HTTP API, implemented in Python using Twisted and QT.
